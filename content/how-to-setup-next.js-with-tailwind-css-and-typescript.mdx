Install [Node.js](https://nodejs.org/en/download) if you haven't already.

1. Go to this [GitHub repository](https://github.com/shamwela/next.js-typescript-tailwind-css-starter). I prepared essential packages for Next.js, TypeScript, and Tailwind CSS in that repository.
1. Click the green "Code" button.
1. Click "Download ZIP".
1. Unzip the downloaded zip file.
1. Rename the folder to your project name.
1. Open the folder in the terminal.
1. Run `npm i`.
1. Now, you can start coding.
