# How to setup Next.js with Tailwind CSS and TypeScript

To create a project with Next.js and Tailwind CSS, run the following commands. TypeScript will be added later.

```shell
npx create-next-app -e with-tailwindcss your-project-name
cd your-project-name
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
```

In the root of your project folder, create a folder called `styles`. Inside this folder, create a file called `globals.css` and add the following code:

```css
/* ./styles/globals.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
```

In `pages/_app.js` file, import `globals.css`:

```jsx
// pages/_app.js
import 'tailwindcss/tailwind.css'
import 'styles/globals.css'

function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />
}

export default MyApp
```

In the root of your project folder, create a file called `tsconfig.json`. And run the following command:

```shell
npm install --save-dev typescript @types/react @types/node
```

Add baseUrl for easy import statements. For example instead of writing `import '../styles/globals.css`, you can write `import 'styles/globals.css'`.

```json
// tsconfig.json
{
  "compilerOptions": {
    "baseUrl": ".",
...
```

Change `.js` extensions to `.tsx`. And start the development server (`npm run dev`). After starting the server, Next.js will:

1. Populate the `tsconfig.json` file for you. You may customize this file.
1. Create the `next-env.d.ts` file, which ensures Next.js types are picked up by the TypeScript compiler. You should not touch this file.

## Optional

In `pages` folder, create `_document.tsx` and add the following code to improve SEO.

```tsx
// _document.tsx
import Document, { Html, Head, Main, NextScript } from 'next/document'

class MyDocument extends Document {
  render() {
    return (
      <Html lang='en'>
        <Head />
        <body>
          <Main />
          <NextScript />
        </body>
      </Html>
    )
  }
}

export default MyDocument
```

Run `npx next lint` and select `strict` to check for linting errors.
