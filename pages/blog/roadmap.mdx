# Web Development Roadmap for Beginners

Teaching yourself how to code is great but one major drawback is that you don't have a strict curriculum like a university. So, I created this roadmap to help you.

Please learn these in this specific order. Don't skip.

## Basics

1. Learn how to type fast
2. HTML
3. CSS
4. JavaScript
5. Git and GitHub

Choose either front-end or back-end.

## Front-end

1. React
2. Next.js
3. TypeScript
4. Tailwind CSS

## Back-end

1. PostgreSQL
2. MongoDB
3. Node.js
4. APIs
5. GraphQL

_Keep learning._
