# How to host a website for free

This will take you about 15 minutes.

[Install Git](https://git-scm.com/downloads)

[Create a GitHub account](https://github.com/signup)

[Create a Vercel account with the GitHub account](https://vercel.com/signup)

[Create a new repository on GitHub](https://github.com/new)

Create a folder on your computer and add some code.

Open your project folder in the terminal and type `git init`

Go back to your GitHub repository. Click on green **Code** button and copy the link. And, type in the following 3 commands.

```
git remote add origin <the link you copied>

Example: git remote add origin https://github.com/yourname/yourproject.git

git commit -am 'First commit'

git push -u origin main
```

Go to [vercel.com/dashboard](https://vercel.com/dashboard)

Click **New Project**.

Import your repository.

Click **Skip** on **Create a Team**.

Click **Deploy**.

Click **Visit** to visit your website.

You can also edit the domain name by going to **Settings** and then **Domains**. The free domain name will end in **vercel.app**. If you want a custom domain like **yourname.com**, you have to buy it.
